# incredibuild — Research Memory

最后更新: 2026-06-16

- **公司/产品/项目**: Incredibuild (Islo AI沙箱, Build Runner Beta), EngFlow (SOC 2 Type 2), FASTBuild, BuildXL, 腾讯云 yadcc, 龙智 (中国区合作伙伴)
- **重要趋势信号**:
  - **Incredibuild 从编译加速向全生命周期 AI 平台转型** (方向: 产品转型, 强度: high)
  - **安全合规成为商业方案核心卖点** (方向: 技术壁垒, 强度: high, 证据: TSL 1.2, ISO 9001/27001, SOC 2)
  - **FASTBuild 在低带宽场景性能劣于 Incredibuild** (方向: 竞品对比, 强度: medium, 证据: 100Mbps下耗时43.5min vs 13.6min)
  - **部分用户从 Incredibuild 迁移至 FASTBuild+Sharpmake** (方向: 用户迁移, 强度: medium)
  - **C++ 开发者群体快速增长** (方向: 市场需求, 强度: medium)
- **值得长期跟踪的技术方向**: AI 原生开发与构建加速融合、分布式编译安全合规、低带宽环境下的编译性能优化
- **竞品动态**:
  - **EngFlow**: 获专题报道，声称 C++ 构建加速 21 倍，通过 SOC 2 Type 2 审计
  - **FASTBuild**: 开源方案，支持共享缓存，但低带宽场景劣势明显，CMake 集成困难
  - **BuildXL**: 与 Incredibuild 对比讨论，聚焦大型构建加速
  - **腾讯云 yadcc**: 开源 C++ 分布式编译，本期无新动态
