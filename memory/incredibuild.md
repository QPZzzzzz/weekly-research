# incredibuild — Research Memory

### 公司/产品/项目
- **Incredibuild**: 获3500万美元B轮融资，转型AI SDLC平台，推出AI agent安全沙箱，服务22万开发者。
- **EngFlow**: 与tipi.build合作CMake RBE方案，宣称C++构建加速21倍，与Uber合办Meetup聚焦monorepo。
- **FASTBuild**: 缓存全命中场景下性能优于Incredibuild。

- **亿道电子/龙智**: 中国代理商，汽车电子客户编译时间缩短50-70%。
