# incredibuild — Research Memory

最后更新: 2026-07-21

- **Incredibuild**：战略转向AI平台，发布AI沙箱Islo（2026年5月），定位为AI编码代理执行层；推出免费CI加速计划（8倍）；停止支持旧版本（9.X及更早）。
- **EngFlow**：新兴商业竞品，声称21倍加速并强调安全，直接对标Incredibuild。
- **开源方案**：FASTBuild（Unity社区推荐）、腾讯yadcc（GitHub活跃，性能优于ccache/distcc）成为免费替代品，威胁扩散。
- **社区负面反馈**：用户抱怨骚扰式销售、认证问题；对替代品兴趣增加，品牌忠诚度面临挑战。
- **竞品动态**：Incredibuild在构建自动化类别中市场份额极低，主要对手为通用CI/CD工具（如Azure DevOps、Jenkins）和专用分布式编译工具（EngFlow、FASTBuild、yadcc）。
- **值得长期跟踪**：AI与构建加速融合趋势（Incredibuild Islo）；开源方案（FASTBuild、yadcc）社区活跃度及性能突破；EngFlow市场渗透率及用户反馈变化。
