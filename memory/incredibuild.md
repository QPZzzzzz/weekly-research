# incredibuild — Research Memory

最后更新: 2026-06-27

- **公司/产品**: Incredibuild, EngFlow, 腾讯 yadcc, FASTBuild, Bazel, Jenkins, TeamCity, GitLab, CircleCI, BuildXL
- **趋势信号**:
    - **Incredibuild 转型 AI 原生平台** (high): 发布 Islo AI 沙箱和 Build Runner，从编译加速转向 AI 开发平台。
    - **C++26 标准增加编译复杂度** (high): 推动分布式编译从“可选”变为“刚需”，利好全行业。
    - **低带宽性能是商业产品核心壁垒** (medium): Incredibuild 在 100Mbps 下性能是 FASTBuild 的 3 倍以上。
    - **EngFlow 宣称 C++ 构建速度提升 21 倍** (medium): 通过性能和安全认证（SOC 2）正面挑战 Incredibuild。
    - **Incredibuild 用户规模数据公开** (high): 22 万开发者、600+ 企业、9400 万次构建，证明市场领导地位。
    - **Incredibuild 用户反馈稳定性问题** (low): 可能成为竞品攻击的弱点。
    - **Incredibuild 获得 ISO 9001/27001 认证** (medium): 增强在金融、医疗等合规行业的竞争力。
- **长期跟踪方向**: AI 原生开发平台演进、C++26 标准对编译生态的影响、低带宽场景下的分布式编译技术、EngFlow 性能宣称的第三方验证。
- **竞品动态**:
    - **EngFlow**: 通过“21 倍速度提升”和 SOC 2 认证，在高端市场与 Incredibuild 正面竞争。
    - **腾讯 yadcc**: 开源替代方案，性能优于 distcc/icecream，对成本敏感用户构成威胁。
    - **FASTBuild**: 开源方案，低带宽性能短板被公开讨论。
    - **Bazel, Jenkins, TeamCity**: 被列为广泛竞品，市场选型关注度提升。
