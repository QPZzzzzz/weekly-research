# incredibuild — Research Memory

最后更新: 2026-06-09

### 关键记忆点

- **公司/产品/项目：** Incredibuild、EngFlow、龙智、FASTBuild、distcc、Bazel、AWS、a16z、Tiger Global
- **重要趋势信号：**
    - **Incredibuild AI 平台发布**：从“编译加速”转向“开发全生命周期”平台，已服务22万开发者，加速9400万次构建。**强度：high**
    - **EngFlow 强势崛起**：由 Bazel 核心团队创立，获 a16z 和 Tiger Global 投资，与 AWS 深度合作，实现 C++ 构建 21 倍加速。**强度：high**
    - **安全合规成准入门槛**：ISO 27001 认证和 SBOM 生成能力成为企业选型关键，是开源方案短板。**强度：high**
    - **构建缓存与预测执行成行业共识**：从“堆算力”转向“提效率”，两大头部玩家均将其作为核心技术。**强度：high**
    - **云原生远程执行方案普及**：EngFlow 与 AWS 合作推动“云原生远程执行”成为大型团队标配。**强度：medium**
- **值得长期跟踪的技术方向/话题：**
    - **AI 全生命周期平台**：Incredibuild 转型方向，关注其平台化能力与市场接受度。
    - **云原生远程执行**：EngFlow 与 AWS 合作模式，是否成为主流部署方式。
    - **Bazel 生态**：EngFlow 的技术基因，关注 Bazel 在分布式编译领域的渗透。
    - **安全合规（ISO 27001, SBOM）**：企业选型硬门槛，关注其他竞品跟进情况。
- **竞品动态：**
    - **EngFlow**：新产品（C++ 构建 21 倍加速）、融资（a16z, Tiger Global）、合作（AWS 远程执行方案）。
    - **Incredibuild**：新产品（AI 平台）、迭代（10.35.x 版本优化 VS 集成）、合作（龙智深耕中国市场）。
    - **开源方案（FASTBuild, distcc）**：持续存在但企业级支持不足，影响力减弱。
