# incredibuild — Research Memory

最后更新: 2026-06-07

- **公司/产品/项目**: Incredibuild (AI平台、10.31.1版)、腾讯 yadcc、微软 Visual Studio、龙智 (中国合作伙伴)、FASTBuild、Bazel、distcc、BuildXL。
- **重要趋势信号**:
    - **Incredibuild AI平台发布**: 从编译工具转型为覆盖全生命周期的AI平台，基于缓存、预测执行和分布式计算，已有22万用户。 (high)
    - **构建缓存与预测执行成核心**: 10.31.1版新增配置集、API、缓存命中详情和预测执行，从“算力分发”转向“数据复用”与“智能调度”。 (high)
    - **安全合规成选型关键门槛**: Incredibuild展示ISO 9001/27001认证和SBOM生成能力，构建信任壁垒。 (high)
    - **腾讯开源yadcc**: 性能优于distcc和icecream，为开源社区提供新选择，可能分流成本敏感用户。 (medium)
    - **微软生态深度绑定**: Visual Studio 17.6 Preview 3深度集成Incredibuild 10，巩固其在VS用户群中的优势。 (medium)
- **值得长期跟踪的技术方向/话题**:
    - AI全生命周期平台（Incredibuild转型方向）
    - 分布式编译的缓存与预测执行技术
    - 软件供应链安全（SBOM、ISO认证）
    - 开源分布式编译器（yadcc、FASTBuild、Bazel）的社区活跃度与企业级支持
    - 大型C++项目的构建性能优化
- **竞品动态**:
    - **新产品/开源**: 腾讯开源yadcc分布式C++编译器。
    - **融资/合作**: 微软Visual Studio深度集成Incredibuild 10；Incredibuild通过中国合作伙伴龙智进行本地化推广。
    - **技术突破**: Incredibuild发布AI全生命周期平台；10.31.1版新增配置集、API管理、缓存命中详情和预测执行功能。
